Methodology

2D Sprites

Programmatic character frames with animation variants (idle, walk, attack, jump, cast, shoot), packed into sprite sheets with JSON metadata for game engine integration. GIMP Script-Fu scripts generated for professional batch processing.

Procedural Textures

Six texture types (stone, wood, metal, fabric, brick, sand) generated algorithmically at 256/512/1024px, packed into optimized atlases with UV coordinates. GIMP Python-Fu scripts for filter chains.

Vector UI Design

Inkscape SVG elements with proper layer structure, gradients, glows, and typography. Health/mana/XP bars, inventory slots, dialog boxes, minimap, tooltips. Batch-exportable at 72/144/288 DPI via CLI.

3D Parametric Modeling

Blender Python scripts generating game objects (gems, potions, weapons, shields) with configurable parameters. PBR materials via Principled BSDF. Smart UV Project with island margins.

Usage Examples

Blender (Headless)

blender --background --python output/scripts/blender/model_gem_ruby.py

GIMP (Batch)

gimp -i -b '(script-fu-load "output/scripts/gimp/knight_sprites.scm")' -b '(gimp-quit 0)'

Inkscape (CLI Export)

inkscape --export-filename=output.png --export-dpi=288 health_bar.svg
